--CELL REFERANCING 
Q1.Why do we use cell references instead of typing values directly into formulas?
--We use cell references because formulas update automatically when the underlying data changes. This reduces manual work, improves accuracy, and makes formulas reusable across many rows.

Q2: What is a Relative Cell Reference?
-- A Relative Cell Reference automatically changes its row and column references when a formula is copied or moved to another cell. It is the default type of reference in Excel and is commonly used when performing the same calculation across multiple rows or columns.

Q3: When should you use an Absolute Cell Reference?
-- An Absolute Cell Reference is used when a formula should always refer to the same cell, even after copying it to other locations. It is commonly used for constants such as tax rates, discount percentages, exchange rates, and commission values.
