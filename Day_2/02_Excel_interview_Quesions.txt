📚 Mini Quiz (Very Important)

Before we move to 3-condition SUMIFS, answer these.

Q1 Which function should you use?
Manager asks:
Total sales of Laptop products.
Choose:
SUM
SUMIF
SUMIFS

Explain why.
- we can use sum function but for better and realistic problems we will use sumifs() function for it and here sum can't be used because it will sum the whole columns sales price

Q2

Manager asks:

Total sales of Laptop products in the North region.

Choose:

SUM
SUMIF
SUMIFS

Explain why.

- here we have only one option to use because here we have two conditions which can be solved using sumifs()
